Để kết nối được với server, cần có 1 số thao tác sau:
+B1:
	Trong trang web phpMyAdmin tạo 1 database mới như sau:
		chọn New -> Nhập databse name là cinema -> Chọn Create
+B2:
	Truy cập vào file Connection.php theo đường dẫn: ~\WebFinal\php\Roles\Connection.php
+B3:
	Nhập servername, username, password tương ứng của XAMPP vào 3 biến $servername, $username, $password.
+B4: 
	Hoàn tất, truy cập localhost để mở trang web
